This program requires Maven 3.5.4 or above.

Please clone the repository from GitHub and navigate to the root folder where pom.xml exists.
The Car Rental Rest Service runs as a Spring Boot app at port 8090.

To start the service, please run below command from the root directory where pom.xml is located-

mvn sprint-boot:run

The three questions are implemented in a JUnit test.
To run the tests please run the below command-

mvn test
